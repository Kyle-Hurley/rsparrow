Getting Started with rsparrow

Introduction

rsparrow is an R package for executing and evaluating SPARROW (SPAtially Referenced Regressions On Watershed attributes) models. It generates estimation results, predictions, and diagnostic output for static (long-term mean annual) SPARROW water-quality models.

Static SPARROW models provide reach-level predictions (and uncertainties) of the long-term mean annual water-quality loads and flow-weighted concentrations. The predictions include the shares of the load attributable to individual upstream sources and predictions of the mass quantities of the total load and individual sources that are delivered to downstream water bodies.

Installation

Install rsparrow from CRAN:

install.packages("rsparrow")

Or install the development version from GitHub:

# install.packages("remotes")
remotes::install_github("Kyle-Hurley/rsparrow")

Basic Usage

library(rsparrow)

# Run a SPARROW model (requires a configured model directory)
model <- rsparrow_model(path_main = "path/to/model/", run_id = "myrun")

# View a summary
summary(model)

# Generate predictions
model <- predict(model)

# Scenario analysis: reduce a source by 50 percent
model <- rsparrow_scenario(
  model,
  source_changes = list(fertilizer = 0.5),
  scenario_name  = "reduce_fertilizer"
)

Model Directory Setup

A SPARROW model requires a directory containing:

  • sparrow_control.R — control settings script
  • parameters.csv — parameter configuration
  • design_matrix.csv — source-delivery design matrix
  • dataDictionary.csv — variable metadata
  • data1.csv — reach-level input data

See the package documentation (?rsparrow_model) and the USGS RSPARROW documentation for detailed setup instructions.
